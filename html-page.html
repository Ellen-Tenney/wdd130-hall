Learn HTML Basics

What is HTML?

HTML stands for HyperText Markup Language. It's the standard language used to create and structure content on the web. Think of HTML as the skeleton of a website - it provides the structure and content, while CSS (which you'll learn later) makes it look pretty!

HTML uses "tags" to mark up different parts of your content, telling the browser how to display text, images, links, and other elements.

Basic HTML Structure

Every HTML document follows a basic structure. Here's what a simple HTML page looks like:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello World!</h1> <p>This is my first webpage.</p> </body> </html>

Breaking It Down:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element that contains all other elements
  • <head> - Contains metadata about the document (title, character set, etc.)
  • <body> - Contains all the visible content of the webpage

Common HTML Tags

Here are the most important HTML tags you'll use regularly:

<h1> to <h6>

Headings - h1 is largest, h6 is smallest

<p>

Paragraph - for blocks of text

<a>

Anchor - creates hyperlinks

<img>

Image - displays pictures

<div>

Division - a container for grouping elements

<ul> & <ol>

Lists - unordered (bullets) and ordered (numbered)

<strong>

Makes text bold and important

<em>

Emphasizes text (usually italic)

Practical Examples

Creating a Link

<a href="https://www.google.com">Click here to visit Google</a>
Output: Click here to visit Google

Adding an Image

<img src="images/photo.jpg" alt="Description of image">

Note: The "alt" attribute describes the image for accessibility and appears if the image fails to load.

Creating a List

<ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul>
Output:
  • First item
  • Second item
  • Third item

HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Common Attributes:

  • href - Specifies the URL for links (<a> tags)
  • src - Specifies the source file for images, scripts, etc.
  • alt - Provides alternative text for images
  • class - Assigns a class name for CSS styling
  • id - Gives a unique identifier to an element
  • style - Applies inline CSS styling
<img src="photo.jpg" alt="My photo" class="profile-pic" id="main-image">

  • Practice regularly: The best way to learn HTML is to build actual web pages!

Next Steps in Your HTML Journey

Now that you understand the basics of HTML, here are some topics to explore next:

  1. Semantic HTML - Using meaningful tags like <article>, <section>, <nav>
  2. Forms - Creating interactive forms with input fields, buttons, and more
  3. Tables - Organizing data in rows and columns
  4. HTML5 Features - Video, audio, canvas, and other modern elements
  5. Accessibility - Making your websites usable for everyone

Remember, HTML works hand-in-hand with CSS (for styling) and JavaScript (for interactivity) to create complete, modern websites!
